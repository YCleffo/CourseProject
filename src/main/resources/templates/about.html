О приложении

Система управления фильмами

Веб-приложение для управления базой данных фильмов и актёров.

Функциональные возможности:
  • Аутентификация и авторизация пользователей
  • Разграничение доступа по ролям (READ_ONLY, USER, ADMIN)
  • Управление фильмами и актёрами
  • Расчёт прибыли фильмов
  • Ведение логов в отдельный файл
  • Мониторинг через Spring Boot Actuator
Используемые технологии:
  • Spring Boot
  • Spring Security
  • Spring Data JPA
  • MySQL
  • Thymeleaf
  • Bootstrap 5
  • Lombok
